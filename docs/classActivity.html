Detailed Description

Representa uma entidade Activity composta por um código, nome, data, horário, duração, preço e avaliação.

A classe Activity encapsula informações sobre uma Atividade, como código, nome, data, horário, duração, preço e avaliação. Permite a manipulação e retorno dessas informações através dos métodos get e set.
