Detailed Description

Representa uma classe de unidade de teste específico para entidades.

A classe EntityUnitTest herda diretamente de UnitTest usando sua interface e implementado propriedades e uma forma de test próprio, através do recebimento de um dado domínio e o nome da propriedade associada para testar a entidade.
