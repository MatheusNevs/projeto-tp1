Detailed Description

Representa uma classe de unidade de teste DomainUnitTest específico para teste de domínios.

A classe DomainUnitTest herda diretamente de UnitTest usando sua interface e implementado propriedades e uma forma de test próprio, através de um valor valído e um valor inválido referente ao domínio específico.
