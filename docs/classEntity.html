Detailed Description

Representa uma classe entidade abstrata Entity para as entidades se construirem sobre.

A classe Entity não possui propriedades próprias, as entidades respectivas que implementarão as propriedades. Possui métodos abstratos 'set' e 'get' que irão ser usados pelas entidades, cada uma com sua própria implementação.
